Sobre o PLAME

Este é apenas um projeto para estudo e desenvolvimento de habilidades nas linguagens de programação incluindo HTML, CSS, Javascript. O emulador utilizado neste projeto foi o NESBOX, ele executa a emulação de todos os consoles apresentados no site até o momento. Utilizei ROMS dos jogos os quais tenho os cartuchos. O site não tem nenhum fim além do estudo e desenvolvimento de habilidades.

Notas:

Este site foi feito utilizando o Visual Studio Code. Sua formatação foi pensada para computadores, pois é necessário teclado para poder jogar, portanto a formatação para celular pode não ser ideal.

Para o futuro:

  • Sistema de Cadastro de Usuários Utilizando SQL
  • Sistema de Saves para Usuários Cadastrados (Manter os 'Save States' de cada usuário, para cada jogo, salvo em servidor, de maneira que ele possa logar e continuar cada um de seus jogos de onde salvou)
  • Ranking dos Players (Exemplo: quem terminou o jogo mais rápido, com maior score, etc)
  • Desenvolver Plataforma Multiplayer via Web(A ideia é ser possivel por exemplo, jogar street fighter contra outro usuário do site, assim será possivel desenvolver campeonatos e competições, além dos usuários terem liberdade de jogar com outros de acordo com sua vontade. Atualmente só é possivel multiplayer no próprio local)
  • Inserir novas plataformas com Playstation 1 e Nintendo 64
